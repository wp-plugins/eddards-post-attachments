Eddard's post attachment plugin makes post attachement handling slightly easier and help you don't loose your head.

== Description ==

This plugin has limited features but makes post attachment images easily accessible while you are editing a post or page.
It has a separate button inside the post editor to open up an additional window for images. There you can access all 
images attached to the current post or images attached to another posts and all unattached images if necessary.
With Eddard's Post Attachment plugin you can also detach attached images from the current post or attach an unattached image
to the current post.

= Translations =

* English
* Hungarian


== Installation ==

1. Upload the entire `eddard-post-attachments` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
